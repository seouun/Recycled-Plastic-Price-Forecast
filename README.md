# 시계열자료분석: 재활용 폐플라스틱 가격 예측

📌 2003년 1월 ~ 2025년 1월까지의 국내 재활용 폐플라스틱 가격 데이터를 활용하여  
시계열 모형(ARIMA 계열)을 적용하고 향후 가격을 예측한 프로젝트
본 연구는 자원순환경제 및 ESG 경영 관점에서 폐플라스틱 가격 예측의 실질적 시사점을 제공.


## 프로젝트 배경
- 자원순환경제와 ESG(Environmental, Social, Governance)의 부각으로 재활용 자원 가격 분석의 중요성 증대
- 플라스틱은 사용량은 많지만 폐기·재활용 과정에서 환경적 부담이 큼
- 재활용 폐플라스틱 가격 변동은 정책 효과성과 순환경제 활성화 수준을 가늠할 수 있는 척도
- 따라서 시계열 분석을 통해 가격 변동 패턴을 이해하고, 향후 추세를 예측하는 것을 목표로 함 


## 데이터
- **출처**: 국내 재활용 폐플라스틱 평균 가격 데이터
- **기간**: 2003.01 ~ 2025.01 (월별, 단위: 원/kg)
- **특징**: 장기적 상승 추세, 계절성은 약하며 분산은 안정적
- **전처리**: 로그 변환, 1차 차분 → 정상성 확보


## 방법론
1. **EDA**
   - 시계열 추세 확인
   - ACF/PACF 분석으로 자기상관 구조 파악
2. **모형 식별**
   - ARIMA(p,d,q) 후보 모형군 도출 (ARIMA(0,1,2), ARIMA(1,1,1), ARIMA(2,1,0) 등)
3. **모형 진단**
   - AIC, Log-Likelihood 기반 적합도 비교
   - 계수 유의성 검정(Z-test)
   - 잔차 진단 (Ljung-Box 검정 → 백색잡음 여부 확인)
4. **예측**
   - 최종 모형 기반 2025년 2월 ~ 12월 가격 예측 수행 


## 주요 결과
- **최적 모형**: ARIMA(0,1,2), ARIMA(2,1,0)  
  - 두 모형 모두 통계적 적합성 충족
  - ARIMA(0,1,2): 간결한 구조, 모든 계수 유의미
  - ARIMA(2,1,0): 장기 예측에서 미세한 변동성을 더 잘 반영
- **예측 결과**: 2025년 플라스틱 가격은 **완만한 상승세** 예상  
- **시사점**:
  - 재활용 원가 상승 및 자원 수급 구조 변화 가능성 반영
  - 순환경제 정책 효과 평가에 실증적 근거 제공 


## 결론
- 본 연구는 재활용 폐플라스틱 가격의 **장기적 추세와 예측 가능성**을 검증
- 정책·산업 측면에서 **자원 관리 및 순환경제 실현**에 기여할 수 있는 전략적 통찰 제공
- 향후 연구: 외생변수(국제 유가, 환율 등) 포함한 확장 모형, LSTM·Prophet 등 딥러닝 모델과의 비교 실험
